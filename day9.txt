==> dot notation is use to access the static keys like id , age , username , address 
==> bracket notation is used when you create dynamic key
  n JavaScript, a dynamic key means using a variable or expression as the name of an object
  property instead of writing the property name directly.

1. Using computed property names (recommended)
const key = "name";

const person = {
  [key]: "Alice"
};

console.log(person);
// { name: "Alice" }

The square brackets [] tell JavaScript to evaluate the expression and use 
its value as the property name.


We can add new properties in over object 

let s = {
    name : "k",
    id : 12,
}
s.address = "limbda";
consol.log(s); // {name : "k" , id : 12 , address : "linbda"};


let s = {
    name : "k",
    id : 12,
}
s.address = "limbda";
s["k"] = "kk";
console.log(s); //{ name: 'k', id: 12, address: 'limbda', k: 'kk' }
